Para que serve?

A estrutura de repetição possibilita a

execução repetida de um código contanto que uma condição

seja verdadeira.

Qual o diferencial?

O Do While executa os comandos ao menos uma única vez, mesmo que a

condição seja falsa desde o início.

O loop infinito ocorre quando a condição para que o loop pare nunca for atingida. Para evitar que isso aconteça, uma técnica útil é estabelecer um limite máximo de iterações. Assim, mesmo que a condição para sair do loop não seja atingida, o loop terminará após um número específico de iterações, evitando que o programa fique preso indefinidamente.
